HEXOVOLANTIS教程VOLANTIS

发布于：2021年2月22日

更新于：2021年2月25日

次浏览

字数：6.1k字

时长：34分钟

在接触了yilia，next和volantis之后，最终选择了volantis。
选择volantis有以下几点原因：

  • 颜值高
  • 易于上手
  • 官方文档写的很详细

个性化路上参考了几位大佬的博客，他们分别是：

  • TRHX’S BLOG
  • AnFrank’s Blog
  • XU’S BLOG
  • 黑石博客
  • 阳光派PLUS
  • 好好学习的郝
  • 荷塘月色

其中，黑石放出了自己魔改的volantis版本，而XU的评论区干货很多，一定要看。

建议

  • 务必详细阅读整个Volantis官方文档。
  • 最好在博客站点根目录下新建一个_config.volantis.yml文件，将themes/volantis/_config.yml中的内容复制过来，之后在新的_config.volantis.yml中修改就好。
  • 博客更应该注重内容，样式只要看着不别扭就行，没必要整的花里胡哨的。
  • 站点配置文件_config.yml位于博客根目录中也就是一开始hexo所处的文件夹。
  • 主题配置文件_config.yml位于themes/你的主题名字/中。

具体安装，部署之类的我就不说了。网上随便找找都有（小声bb，虽然csdn上有很多质量博文，但是更多的博文简直不能看，全是copy-ninja）
我也不知道从哪里写，索性就按volantis的模块一个个说吧。

1.5 标题制作

我的标题制作使用ppt实现的。效果如图：

  • 首先下载王羲之字体，并安装，如何安装请自行百度。
  • 进入ppt软件中画一个文本框，里面写你的标题，字体设置为王羲之字体。
  • 修改样式。
    • 调整字体大小，我的是选择最大的。
    • 给每个字上不同的颜色。
    • 设置文本框背景色和边框无色。
    • 在文本框上右键，选择另存为图片即可。

1.7 crash_cheat

效果如图：


就是一个装饰用的，实现方法如下：

  • 在themes\volantis\source\js下新建一个js文件，crash_cheat.js。代码如下：

      $(function () {
    var OriginTitle = document.title;
    var titleTime;
    document.addEventListener('visibilitychange', function () {
      // 解决pjax冲突bug
      if (document.title !== '╰（‵□′）╯ 我是！') {
          OriginTitle = document.title
      }
      if (document.hidden) {
          $('[rel="icon"]').attr('href', "https://i.loli.net/2021/02/22/Ej89S5cqwQzhT6V.png");
          $('[rel="shortcut icon"]').attr('href', "https://i.loli.net/2021/02/22/Ej89S5cqwQzhT6V.png");
          document.title = '╰（‵□′）╯ 我是！';
          clearTimeout(titleTime);
      } else {
          $('[rel="icon"]').attr('href', "https://i.loli.net/2021/02/22/aZhi1wStPpoYFm6.png");
          $('[rel="shortcut icon"]').attr('href', "https://i.loli.net/2021/02/22/aZhi1wStPpoYFm6.png");
          document.title = '(╯‵□′)╯︵┻━┻ 火车王！ ';
          titleTime = setTimeout(function () {
              document.title = OriginTitle;
          }, 2000);
      }
    });
 })

  • 引用js。
    在\themes\volantis\layout_partial\scripts文件夹中的index.ejs中的最后一行添加以下代码。

    <script src="/js/crash_cheat.js"></script>

  • 你可以按照喜好，自定义上述代码中的title和href部分。
